## в—Ҳ Getting Started

### Prerequisites

```bash
node --version   # v18+ recommended
npm --version    # v9+
```

### Clone & Install

```bash
# Clone the repository
git clone https://github.com/Mazennaji/VueJS-Portfolio.git

# Navigate into the project
cd VueJS-Portfolio

# Install dependencies
npm install
```

### Development

```bash
npm run dev
```

рҹҢҗ Open [http://localhost:5173](http://localhost:5173) in your browser.

### Production Build

```bash
# Build for production
npm run build

# Preview the production build locally
npm run preview
```

Output lands in `dist/` вҖ” ready to deploy anywhere.
